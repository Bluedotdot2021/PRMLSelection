## 项目介绍
本项目是PRML Page-by-page项目的补充。考虑到PRML Page-by-page总共有超100个视频，总时长超65小时，内容量巨大，因此从整个项目中精选了一部分重点问题作快讲。

		   2-059，指数族分布

Chapter4： 4-046，拉普拉斯近似  
Chapter5： 5-027，BP算法  
Chapter6： 6-010，非向量化数据的核函数  
		   6-016，空间大集合  
Chapter7： 无

           8-019~8-024，条件独立性
